Mobile Legends

Mobile Legends is a MMORPG mobile game, that consist of many hero character on which you need to fight using brain tactics and
find a way to kill and destroy the enemy base. At the start of the game, each player gets to select a hero. Each hero has a unique set of attributes and skills.
